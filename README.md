# 🧠 Skin Cancer Detection System

## 🔍 Overview

This project is a machine learning-based application that detects whether a skin lesion is **benign or malignant** using image classification techniques.

---

## 🚀 Features

* Image-based skin cancer detection
* Machine learning model prediction
* Fast and efficient results
* Simple and easy-to-use system

---

## 🛠️ Technologies Used

* Python
* TensorFlow / Keras
* OpenCV
* NumPy
* Pandas

---

---

## 📂 Project Structure

```
skin-cancer-detection/
├── train_model.py
├── predict.py
├── requirements.txt
└── README.md
```

---

## ▶️ How to Run

1. Clone the repository

```
git clone https://github.com/your-username/skin-cancer-detection.git
```

2. Navigate to project folder

```
cd skin-cancer-detection
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the model

```
python predict.py
```
